Submit Assignment Notebook Using Pull Request

1. Fork and Clone the repository owned by earthlab-education 

Recall that after you accept your assignment using the link above, GitHub will create an Assignment  repository owned by earthlab-education (e.g. https://github.com/earthlab-education/bootcamp-2020-08-loops-your-user-name-here).

Fork this repository on GitHub.com so that you own a copy of it.  (e.g. https://github.com/your-user-name/bootcamp-2020-08-loops-your-user-name).

Then, git clone your fork the files to your local working environment (e.g. local computer or JupyterHub). 

2. Complete assignment notebook

Use git add, git commit, and git push origin master to send the changes to your fork on GitHub.com.

3. Submit a Pull Request

Submit a pull request from your fork to the repository owned by earthlab-education

In the pull request, include the following: 

  1. a succinct but descriptive title (i.e. the owner of the repository can easily tell what was completed in the pull request without needing to read the pull request comment)

  2. additional details in the comment to describe what was completed in this pull request

4. Merge Your Pull Request

When you are ready to submit your assignment, you will merge your own pull request from your fork into the main repository owned by earthlab-education.
